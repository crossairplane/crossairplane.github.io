落网记忆

发表于 2019-11-08 | 更新于 2019-11-09 | 分类于 音乐 | | 阅读次数： | 阅读次数：

「独立，不独于世」这句话是落网的slogan，而「落网」又是什么呢？这里有介绍:是一个自03年起坚持了十几年的独立音乐网站，2018年末陆续打不开，至今官方网站再也打不开的小站。如同大多数互联网世界里的故事一样，终于曲终人已散，流星再灿烂也已是从前。而从前的日子像是一场独行侠们深夜的狂欢，只留下无数落友「等你回来的期盼」。

不过，其实没那么伤感，当代互联网的骑士精神永存，落网记忆是今年8月份网友们做的存档网站，落网之前的期刊、电台、文字都在，只是评论没了，用户账号不存在了。10月发现这个存档站是我最开心的事！

其实，想聊聊落网已经很久了，不过就算是现在也好像没办法准确地说出为什么我喜欢它，也无法分辨出我是喜欢它的音乐还是它本身，不过，我想这已经不重要了，就来梳理一些关于落网的记忆，也是关于从前日子的回忆。

转着圈圈的那个下午

2013年某一天的下午，忘记怎么突然点开的链接，第一次来到还只有网页端的落网，那一期是《激荡的人生当如此》，第一次听到了John Jett的 I hate myself for loving you而且还是现场版本，有种初听即巅峰的感受——那个下午突然变得闪闪发光，我站在床边戴着耳机听了一遍又一遍，平淡如水的生活顿时被牛逼轰轰的什么东西炸出星火，第一次感受到了音乐带来的震撼，从此喜欢上了摇滚乐，整个人在那个下午感觉都在旋转，好像随时就要起飞，轰轰隆隆、晕晕乎乎、感受到了新世界。

一万首歌的故事

渐渐的，晚上睡前去落网听歌变成了习惯，很快就到了10000首累计播放量，我也从这个不定期更新、坚持人工选歌的独立网站里，获得很多温暖。我在那里知道了李志、尹吾、脑浊、高旗超载、鲍家街43号、mother mother、my little airport、new medicine、sek laso、mazzy star、keane、placebo……而时间不知不觉就去了很久、很久。

有一期是Time，我每隔一阵子就会因为收到听友赞的消息提醒而去留留言，成了独特的记录：从2014到2017，好像歌曲量的数字叠加不停在流逝的日子就是彼时的心情，「Time过去过去、来了来了」。

众筹是什么愁

2016年起，落网安卓、塞班、苹果都有了客户端，团队规模空前庞大，付费节目上线、实体落网live house广州落成，风风火火，最后还是走向了衰落。其实，从一开始版权就是绕不开的硬伤，知乎、搜狐等媒体上的一些文章已经有分析过了，不想再说，作为「白嫖时代的成长起来的音乐迷」，也算是没什么资格站在道德制高点上侃侃而谈。只记得2016年参与落网空间线下店的众筹，周边是好看的落网logo黑色T恤和一张哑光黑色充值卡，特别喜欢那件T恤，印花图样是只黑天鹅，穿出去的时候有人会以为是mazzy star乐队的周边，但落网的logo就是受了这只乐队的影响。不过好景不长，不爱丢东西的我，在某次天台晾晒后再也没找回来，想起来就愁绪范范，伴着落网第二间线下空间经营不善一起，看着这个从前的独立小站的日常运营变得艰难，最后再也打不开，期刊停留在了第999期。

燃尽重生

又是一年快尽时，以往，落网都会出「再见」系列，再见2014、2015、2016、2017、2018，现在是2019年了，二十一世纪里的第二个十年一晃眼也就要过去了：再见了，落网。空虚的尽头是重生，第1000期再会吧。
