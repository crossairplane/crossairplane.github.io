执行力

发表于 2019-09-05 | 分类于 随笔 | | 阅读次数： | 阅读次数：

长期以来，一直有受到「要完成点什么」的压力，不知道源来，但给到的生活体验就是：不自恰。

“Document. Don’t create.”虽然这句话是一直有提醒自己要专注在过程里，把目标分解，做好适时地调整，遵循现在流行的所谓先从0到1，再小步快速迭代来趋近最终结果，但是实际运用中，并没有很好地实践。而正如陈嘉映在《何为良好的生活》一书中谈到的”良好的实践就是良好的生活本身”，当前对于实践的缺乏，又引出了新的隐患：良好的生活怎么去追寻？显然，一连串的问题接踵而至，已经超出了我处理能力的阈值上限，追根溯源：似乎执行力成了第一个蹦出脑海中的关键词，继而又联想到了人们常说的“想的太多而做的太少”即由于生产投入和结果导向之间不匹配引发的矛盾。既然没办法求最优解，那不妨谈谈如何提高执行力。

最近一阵子，对执行力有些良好的实践：
1.尝试着把生活中的一些希望通过比较长的时间投入才能看到结果的事项列为必须.譬如说，如果不加班下班后第一件事就是去运动，这样一来没有选择的日程，极大减少了正在减脂期间的焦虑感，只要去做就好了的念头反而让整件事情没有那么痛苦，关键点在于不用面对选择，心情大好😊。

2.随时都可以开始，也可以在任何时候选择停下。这点其实在我们吃东西，特别是吃零食的时候实践得特别好，但其实是可以迁移到其他任务中去的。最近在同时读多本书籍，而原来我是特别执拗的：如果开始读一本书，就一定要等到读完后再开始下一本，导致要么一本书读很久，要么再也读不下去就束之高阁甚至对阅读产生厌恶；现在保持轻松的心态开始，不想读的时候即刻停止的阅读方式让我开始体会到阅读的乐趣。

3.“Done is better than perfect”，减少拖延。个人是非常反感“一味追求效率至上，生活必须有意”之类的上古鸡汤文学的，保持开放、包容，尊重个体选择，可以的话我倒是觉得「不带自责地散漫度过生命中的闲散时光」是件很酷的事情；但如果「不幸」的无法享受当下的闲暇，尽快开始不要拖延绝对是种救赎（我知道，老生常谈，可是架不住有用啊），这里强调的是：当一次次不停地去积累完成而非完美的信念时，事情的可爱程度会超乎预期。譬如，我现在写的博文就是其中一次，当个人工作体系没有那么有效和足够完善地来支撑起预期产出时，亟需这种「Done」的次数叠加，产生正向激励，才能使事情看来不至于被流逝的时间吊打，不让自己又掉进修正计划的情景，获得按部就班的踏实感。

以上就是最近的一些执行力实践吧，离实践出良好的生活差距甚远，可总比不做强。（再次实践了「Done」🤟）。
