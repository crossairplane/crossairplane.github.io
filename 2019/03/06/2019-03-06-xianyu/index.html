Xian鱼

发表于 2019-03-06 | 更新于 2019-09-05 | 分类于 小诗 | | 阅读次数： | 阅读次数：

醒来，工作，吃饭
然后想到了：xian鱼
「闲鱼」和「咸鱼」最大的不同，
可能是：
一个闲死，
一个咸死。
他们忘记了要在海里
毕竟7秒的记忆，太短
短到忘记了，
为什么会死
怎么就死了？
反正，xian鱼的一天
周而复始，
也走向了终点
最后，明白了：
一条xian鱼的使命，
应当是开始，
在每个7秒结束以后，
重新活过一生。
