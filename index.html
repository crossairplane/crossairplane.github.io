执行力

发表于 2019-09-05 | 分类于 随笔 | | 阅读次数：

长期以来，一直有受到「要完成点什么」的压力，不知道源来，但给到的生活体验就是：不自恰。

“Document. Don’t create.”虽然这句话是一直有提醒自己要专注在过程里，把目标分解，做好适时地调整，遵循现在流行的所谓先从0到1，再小步快速迭代来趋近最终结果，但是实际运用中，并没有很好地实践。而正如陈嘉映在《何为良好的生活》一书中谈到的”良好的实践就是良好的生活本身”，当前对于实践的缺乏，又引出了新的隐患：良好的生活怎么去追寻？显然，一连串的问题接踵而至，已经超出了我处理能力的阈值上限，追根溯源：似乎执行力成了第一个蹦出脑海中的关键词，继而又联想到了人们常说的“想的太多而做的太少”即由于生产投入和结果导向之间不匹配引发的矛盾。既然没办法求最优解，那不妨谈谈如何提高执行力。

乐队和这个夏天

发表于 2019-08-25 | 更新于 2019-09-05 | 分类于 音乐 | | 阅读次数：

《乐队的夏天》是2019年夏天的惊喜，已经很久没有对某个夏天有着深刻的记忆了，特别是与某个节目有关。记忆好像一下子回到了初中，回到了那些年选秀盛行的日子里，一边是吃不完的杨梅、冰棒，一边是给人惊喜的参赛选手。而这个夏天，全是乐队的，好像来的晚了些，晚到Billboard前50都很少有摇滚乐队的歌了，晚到Oasis都解散整整十年了。虽然有Coldplay、the 1975、Maroon 5，Westlife有生之年也再出专辑了，乐队继续，但是，摇滚乐队毫无疑问已经不再流行，伍德斯托克音乐节50周年票滞销，1994年红磡演唱会也成了不可超越的中国摇滚史顶峰：摇滚乐队的夏天早已过去。

但，慢半拍的中国摇滚乐，在2019年，借着互联网和自制综艺的光和热，迎来了迟到的狂欢。一群平均年龄35岁的音乐人，重新唱起十年前的歌，一切都那么复古又新奇，恍惚了时间的界限，开始了Party。
