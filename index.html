关于近期一些言论自由的讨论

发表于 2020-05-01 | 分类于 随笔 | | 阅读次数：

题记：

这是2月13日，我在居家隔离期间，写给高中小妹的一封邮件。当时正值国内新冠疫情严重时期，并没有在全球大规模爆发。现在是5月初了，中国高风险地区为0，在4月26日，武汉在院新冠肺炎患者清零，累计确诊人数8.4万人，死亡4643人；国外累计确诊人数达到了323万人，死亡23万人，其中美国累计确诊人数突破100万人，法国累计确诊人数达到16.7万人。

3月19日，国家监委调查组发布《关于群众反映的涉及李文亮医生有关情况调查的通报》，为李文亮医生事件官方回复，虽然工作建议部分不痛不痒，但我认为是一个进步。

一个健康的社会不该只有一种声音。

新冠病毒肺炎 (COVID-19)全球爆发，两大制度，在疫情防控上都有问题，有各自的优势同时劣势也很突出，回顾这个事件，心态更为平静。所以，也把当时比较「愤青」的文字发出来，无论如何，单纯的感受是会被记忆消退、改变和扭曲的，不看这些文字，仅仅过了2个月，我已经记不起当初短暂的窒息感了，所幸，这封邮件再次阅读的时候，那种深切的感受，真实的想法，被保留，所以，我决定po出来。

原文如下：

最近随着这个疫情的发展，微博等社交媒体，平时可以随便说（其实也不能什么都说么，有审查机制和网络警察的存在，但至少会比现在这个情况相对宽容一些。）邮件相对安全一些，所以想和你说这些最近发生的事，风声雨声读书声，声声入耳；家事国事天下事，事事关心。周恩来总理也说过，要为中华之崛起而读书。虽然到今天令人唏嘘。

一、推崇独立创作

今天在看到Twitter有博主对王志安抱怨Twitter不好用，不能发文章以及长截图一事持有批评看法，表示这是因为中文互联网环境已经让人失去了在网站上发文章再做链接的习惯。对此，虽然我对Twitter不能发长截图的确不满意，但是，同样认同中文互联网环境让「独立」更可贵、也更困难了。

联想到前阵子李荣浩才在微博抱怨自己的歌因为运作太复杂迟迟不能发表，李银河老师的新书宣传有时候会被和谐影响效果，加上这阵子一片一片的404（注：这是一种报错，在网站不能访问时出现，提醒前端程序员某种特定的错误，方便即使修复），不难看出：坚持写作和独立表达，学习并掌握互联网独立建站的能力是值得推行的，任何有需要做表达的人都应该被鼓励去了解一下，这样作家想写什么就写什么，歌手想发表歌曲就去发表，这样的「独立创作者」多了后，头部平台的流量争夺就会更困难，被争夺的流量资源必将是更具价值的；况且，稍微了解一下，在如此丰富的第三方工具面前，能力强的就从0到1，能力稍弱的花一顿饭钱就可以使用这些服务，为什么不尝试一下呢？

此外，我很疑惑的是很多好用的第三方工具的开发者也是国外的，可是国外的互联网环境相比当下中国已经很好了。但细想下：或许正是因为国外对知识产权的重视，对独立表达、言论自由的捍卫才倒逼互联网环境保持开放，所以那些开发者本来就会做这些工具，只是其中几种获得了巨大成功，变成了Twitter、Facebook、YouTube。

那这样做的好处是什么呢？至少李志的音乐这不，还是可以随时想听就听啰：给个链接http://520lo.com（李志云音乐）

再来，就算你服务器挂了，一切灰飞烟没，还是会有痕迹啊，你就是泰戈尔的那只鸟，archive.org会存档你的网站旧照。

二、不吐不快的近期各种诡异评论

前情提要：
1.雷神山、火神山这两座隔离医院建好后，相应的官方微博开通了，这种一般就是官方的网络形象了，然而这两个微博的互动和展现形式就是各种网络用语，各种幼稚化，简直就是娱乐至死的化身，太讽刺了，前线白衣天使们连防护服都没有，这边「宝宝不哭不哭」……

由此，想到GaryVee（注：美国一位优秀的市场营销企业家）评论过：整个互联网世界，整个，持续做创作者的就1%，其他人只是转发、点赞、哈哈哈。说这个不是有什么意见，是不是真的只有1%也不重要，重要的是我觉得我们是不是应该努力下扩大一下所谓的「创作者」的占比呢？这个是值得呼吁的，现在需要人们持续不断的创作来保持说话在「准确、逻辑、朴素」的水平，便于交流，不丢「信、达、雅」更好，也不是让大家放弃梗，我就觉得「天啦噜」明明很可爱啊，但是在好好说话的环境、在需要信息的环境下，放弃练习、放弃坚持输出的后果就是：铺天盖地的「雷神山的雷雷不哭喔，送灰宗加油哟……」。不要以为这种练习不重要，不练就是会退化的，我这种勉强及格的水平就更需要加强了，事实就是如此。我也希望你不要放弃输出，不论是哪种形式的。

4.纪念李文亮医生
不知道你知道李文亮医生吗？他就是那位之前被训诫的“造谣者”，首批说新冠类似SARS，在他的私人朋友圈并不是公开场合让大家多加小心，后来又一直在一线抗疫，最后感染病毒于2月6日 晚上9点过去世的，他就是一位「吹哨人」，但是那天互联网中文世界发生了最严重的封号、删帖事件，原因是有他的同事在微信为他抱不平被截图让网友知道了，爆发了大家集体为李医生要一个说法，还一个公道，撤销他的训诫、造谣罪名的请愿。截图内容是：李医生走后，领导怕舆论压力，让医院医护接着上一种比较昂贵的救助仪器（生前没得用），对尸体「表演式抢救」了3小时，有医疗背景的网友表示这种救助仪器对身体会造成很大损伤，所以这个行为无疑就是鞭尸。就这样，等舆论得到引导，微博“不当言论”控制后，武汉政府发声明说李医生于2月7日逝世，但从头到尾没有承认过他被冤枉，从来没给他一个道歉，没给公众一个答复。但压制的舆论之大还是让中央后续派了调查组过去，但至始至终都不承认错误，删除热搜、减少提及。

电子焦虑

发表于 2020-04-30 | 更新于 2020-05-01 | 分类于 随笔 | | 阅读次数：

下午3点左右突然被叮任务过期，不是什么重要的事，随手把它更改到当下：红色的过期事项一下子变成蓝色待办，看着这样的过渡动画，眼睁睁突然感受到了一丝愕然，我就在想，从什么时候开始，不再为软件或者App感到自我限制？细想，那「自我限制」又是怎么来的呢？

最近两年使用闲鱼平台后，我买卖硬件的频率变高了，一开始是想要极简，东西太多了而自己又没能物尽其用，有种莫名的亏欠感，所幸都卖了回血；另一方面旧的不去新的不来，可以更新到当下的新科技，总是让人开心。然而逃不出「买前生产力，买后XXX」的魔咒，不仅如此，我还感受到了负担，那种令人不快的「搁置」感，逐渐不痛不痒地滋生出一种电子焦虑：焦虑重量，焦虑电量，焦虑App切换，焦虑工具变成全部，甚至让这一切慢慢解构掉「去做」本身。

 Deadline是什么？

发表于 2020-04-09 | 更新于 2020-04-10 | 分类于 随笔 | | 阅读次数：

最近两日好像是「超级月亮」，问了下好友：一个在苏州据说一整天都忘记抬头看天，一个另外的城市表示并不晓得，而我懒得出门——3个人都没有看到月亮！没有一个人拍了月亮。上次我想拍月亮还是某年中秋，当时想着：下次吧。不知不觉，就过去了好几年。

好像，所有没有期限的时间，都是恓惶的承诺。而关于期限，《重庆森林》里有这么一段金城武口白：

从分手的那一天开始，我每天买一罐5月1号到期的凤梨罐头，因为凤梨是阿may最爱吃的东西，而5月1号是我的生日。我告诉我自己，当我买满30罐的时候，她如果还不回来，这段感情就会过期。

年龄来到二十代后半段，开始慢慢感受到人生受锤，经历诚惶诚恐四分之一人生危机，实际上这种感觉持续了快两年，有点慌忙又有点丧，冒牌者症候群时常发作：常常会担忧未来，也终日缺少静下来、沉下去的那份从容，时不时还想要再重来（可是，要是能重来，还要选李白？）。

我，从来没想过会写这样的主题。

然而，工作以来却写了很多。从纪要、通知、通报、公告，到函、请示、申请……每次都让人心慌。也不得不说，虽然不擅长，好歹勉强最后及格了。回顾以往，作为一只自由散漫，罗曼蒂克，执着量词破碎，靠着学习批判性思维方法，才磕磕绊绊地成长为「比较有逻辑」的人，自小就不明白：新闻联播的会议精神传达，到底是如何将一句话讲成30分钟的？终于，通过两年的耳濡目染，我决定为「公文」正名，但是：永远不妨碍我不喜欢它，也认为这不是必要的。
