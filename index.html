那些我2019年买了，但没读完的书

发表于 2020-06-11 | 更新于 2020-06-17 | 分类于 随笔 | | 阅读次数：

本质上我并不是一个喜欢阅读的人，但是喜欢获取信息。获取信息大概分了三个阶段：第一阶段，听广播、看纪录片以及看电影，这时候还是比较被动地接收讯息，方寸间感觉世界都变大了；第二阶段，订阅，具体来说就是有主题地选择信息源，听类型播客、关注视频频道、阅读博客、浏览专业论坛，都属于这个范畴。这个时候算是开始有自主意识地去挖讯息，也获得了两个宝贵的体验以及进步，分别是喜欢上了跑步以及英文听力、阅读水平有了质的飞跃，感受到了自己与世界的联结。第三阶段，我正在尝试，开卷有益似乎是接近于政治正确，也没什么理由去反对的习惯，但是我没有，至今也体会不到阅读的乐趣，可能长到现在读的最多的就是教科书🤷🏻‍♀️。但是，我基本认同「文字是有力量」的，返璞归真，肯花时间写下一本书并经过几次修订，众多人去评价后的好书，生命力是无穷的，好像没有什么理由不去了解一下对吧？

关于近期一些言论自由的讨论

题记：

这是2月13日，我在居家隔离期间，写给高中小妹的一封邮件。当时正值国内新冠疫情严重时期，并没有在全球大规模爆发。现在是5月初了，中国高风险地区为0，在4月26日，武汉在院新冠肺炎患者清零，累计确诊人数8.4万人，死亡4643人；国外累计确诊人数达到了323万人，死亡23万人，其中美国累计确诊人数突破100万人，法国累计确诊人数达到16.7万人。

3月19日，国家监委调查组发布《关于群众反映的涉及李文亮医生有关情况调查的通报》，为李文亮医生事件官方回复，虽然工作建议部分不痛不痒，但我认为是一个进步。

一个健康的社会不该只有一种声音。

新冠病毒肺炎 (COVID-19)全球爆发，两大制度，在疫情防控上都有问题，有各自的优势同时劣势也很突出，回顾这个事件，心态更为平静。所以，也把当时比较「愤青」的文字发出来，无论如何，单纯的感受是会被记忆消退、改变和扭曲的，不看这些文字，仅仅过了2个月，我已经记不起当初短暂的窒息感了，所幸，这封邮件再次阅读的时候，那种深切的感受，真实的想法，被保留，所以，我决定po出来。

电子焦虑

发表于 2020-04-30 | 更新于 2020-05-01 | 分类于 随笔 | | 阅读次数：

下午3点左右突然被叮任务过期，不是什么重要的事，随手把它更改到当下：红色的过期事项一下子变成蓝色待办，看着这样的过渡动画，眼睁睁突然感受到了一丝愕然，我就在想，从什么时候开始，不再为软件或者App感到自我限制？细想，那「自我限制」又是怎么来的呢？

最近两年使用闲鱼平台后，我买卖硬件的频率变高了，一开始是想要极简，东西太多了而自己又没能物尽其用，有种莫名的亏欠感，所幸都卖了回血；另一方面旧的不去新的不来，可以更新到当下的新科技，总是让人开心。然而逃不出「买前生产力，买后XXX」的魔咒，不仅如此，我还感受到了负担，那种令人不快的「搁置」感，逐渐不痛不痒地滋生出一种电子焦虑：焦虑重量，焦虑电量，焦虑App切换，焦虑工具变成全部，甚至让这一切慢慢解构掉「去做」本身。
