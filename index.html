【周粥邹07】期待：新专辑和新书

发表于 2023-02-12 | 更新于 2023-02-19 | 分类于 newsletter | | 阅读次数：

最近突然就能早睡早起了，顺便结束了我长达2年的工作日迟到5-15分钟定律（之所以长达2年，属于是打工人的消极反抗，有点那种「你早就决定好了，要做一种暗淡的金色，时间久了就小小地爆炸」）。

这个转变可能与一直在看一些海南环岛骑行、台湾环岛骑行有关，有些心旷神怡，睡前也会稍微拉伸下。但这个不保证成功，只是个体经验。

🎉本周推荐

🎵音乐
最近一周新专辑，分别来自台湾乐队告五人的《带你飞》、时隔4年再发新专辑的Paramore《This is Why》。告五人的新专辑虽然没有很惊艳，但《我无法克制自己》传唱度还是有的。Paramore整张听完没什么记忆点，时隔几年风格已经变了，听来都是破碎重组又出发的感觉，歌词倒是很好，个人很喜欢《Liar》，写在爱里人们不由自主地脆弱、撕裂、崩溃又渴望被理解，非常细腻的情感，加上Hayley的温柔的声线，很耐听。

值得等待的新专辑：2月24日街头霸王Gorillaz的新专辑发行，先行曲《Skinny Ape》简直就是《On Melancholy Hill》的姊妹篇，来自虚拟卡通乐队满满的关爱，一点点咔叽电子、不多不多少的距离感。期待新专辑！

📚书籍
值得关注的新书：看到Tim Urban最新博文《A short history of my last six years》透露，他终于完成了六年前关于社会问题话题讨论的文章，由于话题太大，从一开始的mini文最终不得不反复增加、删减，最后以书籍形式出版：2月21日发售《What’s our problem? a self-help book for societies》。此外这个6年创作过程的中篇介绍，本身就是一篇好的讲故事的结构：抛出梗，也坦诚讲了自己过于追求完美，以前常用的唤醒恐慌猴子、设置截止日期、众人监督等等技巧都毫无作用，Tim说自己甚至被这种巨大的压力吞噬了，带着这种被萦绕的压力结婚、经历疫情，直到孩子出生前，再再再次立下Flag要做个了解，所以这本书得以出版。

【周粥邹05】阅读的实践方式

发表于 2023-02-06 | 分类于 newsletter | | 阅读次数：

作为一个不怎么阅读书籍的人，常常会听到「你一旦开始阅读，也会永远自由」，目前是没有体会到趋向于自由感的可能性，但不得不说：「自由」非常诱人。

「少」怎么定义？

大概就是在立下Flag之后平均只能每年读10本传记、商业类书籍的数量，不立Flag大概一年都想不起来要读的状态。

而政府公布的阅读数据显示，国人2021年人均纸质图书阅读量为4.76本，人均电子书阅读量为3.3本，有11.9%的国民年均阅读10本及以上纸质图书，有8.7%的国民年均阅读10本及以上电子书。所以，年度人均阅读8本书，如果年度能阅读20本书籍，就是那8.7%了。不禁想想，是大家都卷得没空读书了吗？

【周粥绉04】春天，花，鸟，诗歌

发表于 2023-01-29 | 更新于 2023-01-30 | 分类于 newsletter | | 阅读次数：

春节刚过，复工两天，马上就是24节气里的立春了，那，春天要随机说点春天的事。

杜鹃和杜鹃花的渊源

海子写过春天的诗，很多，其中有一首有些意思：

在春天，野蛮而复仇的海子
就剩这一个，最后一个
这是黑夜的儿子，沉浸于冬天，倾心死亡
不能自拔，热爱着空虚而寒冷的乡村

那里的谷物高高堆起，遮住了窗子
它们一半用于一家六口人的嘴，吃和胃
一半用于农业，他们自己繁殖
大风从东吹到西，从北刮到南，无视黑夜和黎明
你所说的曙光究竟是什么意思

完全没有「面朝大海，春暖花开」的和煦感，有些凄厉，古人有云杜鹃啼血，看来也差不多是这个意思。《迟早更新》的过年特辑里讲到春天有很多鸟过冬归来，其中布谷鸟，也就是大杜鹃，作为春耕的大自然报时钟，与春天花会开且开得非常热烈的杜鹃花（可以感受下：有的地方叫映山红），到底是什么关系？

【周粥邹03】2023零零碎碎的澳网记忆

发表于 2023-01-23 | 分类于 newsletter | | 阅读次数：

澳网

Hi，各位朋友新年好。葵卯年正月初二，今天还有没有继续吃家里的年夜剩饭？😂

本周恰逢2023年澳网，回家几日都在看球，特别是看了几场张帅、穆雷的球，也勾起了比较久远的记忆。江山有待才人出，10年前风华正茂的选手们逐渐成了“老将”。很多同时代的都退役了，还能看到他们的坚持，有些说不出的感动。

安迪·穆雷，这位曾经的四巨头，女权意识和英式幽默ATP第一位的英国男子，2019年澳网伤退，以至于赛事主办方组织各球员送上退役祝福，结果换了金属髋的穆雷四年后还在打球。虽然排名一落千丈，手术后恢复的穆雷从挑战赛开始，真是有什么比赛就打什么，但是移动能力大不如前也是事实，大多止步前两轮，以至于今年澳网一轮就碰见13号种子贝雷蒂尼。在澳网这片球场，穆雷五亚，从年轻打到中年，以前总觉得些许悲情，今年甚至感觉些许悲壮。盘点下穆雷本届赛事签表：第一轮，五盘抢十战胜贝雷蒂尼；第二轮，耗时5小时45分（公开赛年代以来澳网耗时第二）艰难战胜本土双打宠儿科基纳基斯，也算看过凌晨四点的墨尔本🙃；休息4小时不到，开始备赛训练，第三轮，再次面对四年前澳网「退役」之战的对手西班牙阿古特，1-6、7-6、3-6、4-6再次重复剧本败北。早些年就有人说过，穆雷这种防守反击的打法迟早要把自己献祭出去，穆雷倒是说这就是自己的网球哲学。不管是真哲学还是改技术无果，ATP能有这么一位执拗的人，也是风景，何况2016年的安迪·穆雷与教练毛瑞斯莫合作，豪取过24连胜、温网、奥运会双金、法网亚军、年终第一，见过顶峰低谷，还能继续打比赛，这份热爱真独一无二。
