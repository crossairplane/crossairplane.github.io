 去跑步吧

发表于 2020-03-20 | 更新于 2020-03-21 | 分类于 生活 | | 阅读次数：

这些年下来，我确定一件事：我是喜欢跑步的。

从小托家里学体育的舅舅的福，很小开始玩球类运动：小时候看《足球小将》，每天踢足球1小时，周末有空还练习投篮，好像对田径是没有天赋和热情的。后来，可能因为跑步是件门槛很低的事，只要有双合适的鞋，你就可以开始，慢慢成了我唯一的运动。很难说是怎么喜欢上的。

时间轴从大学开始，为了打发时间，每周会去学校操场跑上几圈，标准的体育场馆一圈400米，跑8圈人就要晕，3公里感觉就是长跑了。大概是为了800米能够及格每周坚持跑一次，后来陪室友参加运动会比赛，所以强行撑着练习3公里，大概是每公里10分钟，也没什么进步。不过，我从有意识开始跑步起，就有记录的习惯，那会好像安卓手机有的跑步软件不是很多，用的endomondo，于是有了最早的跑步记录，2013年10月：

中间停了一阵子，时间就到了2015年。我开始注册悦跑圈，跑步的理由依然是无聊，试着买了第一双「跑鞋」，沿着当时居住的房子附近的护城河偶尔跑步，5公里突破性的成绩是40分钟，每公里差不多8分钟。

转折出现在2016年5月，体重一度达到巅峰，估计是当时学业压力空前，陷入无比焦虑的阶段。为减脂，突发奇想每天跑步，白天上课晚上跑，晚上上课清晨6点开始跑，依然是每公里差不多8分钟左右，跑了1个月，体重也没什么变化（饮食没控制），跑了当时最长的里程7公里，配速7’24”/km，全靠公园太美，跑步的时候发现压力没那么大了，特别是出汗后放空什么都不想，望着湖压腿的时候，感受到了安宁。

6月，基本每天7公里左右，完全是错误的示范，幸运的是身体素质还行，跑姿还算没有大问题，膝盖没受伤。一个刚开始跑步的新手，竟然在6月跑了106km，配速突破到6’40”/km。人生第一次体验到「量变到质变」，非常开心。当时开始想起《阿甘正传》，就这样跑到了10月，降了频率，每周2-3次左右，达成第一个10km，跑步彻底成为一种习惯，有空就会跑。

一晃断断续续就到了现在，这几年又出现了比较重要的两个激励点：一个是Casey Neistat，另一个是Run with Casey的活动。2017年2月Casey还在疯狂地日更Vlog，在看完my addiction后，第一次有想把跑步作为一件想一直坚持下去的事，整个视频2分52秒，Casey没有说一句话，只是在跑，屏幕上打出他的跑步宣言：

There are two kinds of people in this world
those who run
and everyone else.

想起他之前在视频里提到过为什么跑步？28岁的时候出了车祸，大腿骨折做了手术，医生告诉他再也不能跑了，所以康复后，他开始每天跑步（不服输，当然也有珍惜的成分）。而当他说每天的时候，是真的每一天，13英里每一天！不过，这倒不是打动我的点，视频最后缓缓打出的一句话，让我产生了共鸣：

Don’t run because you have to
run because you can
run

不久后，另一个视频博主发起了13周run with Casey活动。我也想挑战一下，所以开始固定时间跑步，有意识地训练自己，跑更远或者更快，参加现实生活里家门口的5公里比赛。在跑了比较多个10公里后，2019年下半年第一次给自己定目标：完成半程马拉松，定这个目标的时候，还是更多的受到了当年看《阿甘正传》时突然地自我——就像阿甘一样跑吧，不能穿越美国，至少也要跑个半马。我开始跑15公里、17公里，发现身体能够接受后，我就直接参加了线上的半程马拉松比赛。2019年下半年加入了跑团，共跑了4次20公里；没有抽到正赛签，所以改线上，完成2场比赛。这一年，一并刷新个人所有最佳成绩：5公里一般能在30分钟跑完，最快是28分钟，配速5’47”/km（我知道不算快，当然也不慢）。

从一开始的10分钟每公里到现在，跑步这项自己和自己比的运动，没有直接冲突，非常适契合我的性格。我在跑步的时候可以想非常多的事情，那些平日里解不开的迷，一点一点的有了专门的时间去理线头，跑步成了我想一直做下去的运动。

今年，年初疫情在家看了一部动漫《强风吹拂》，以日本箱根驿传为背景，非常过瘾。看完后抑还制不住激动的心情还发了微博安利，我觉得是为2020年开了好头。跑步的zen好像也变得像是那么一回事儿，于是定下目标全年跑500km，年底见！

一周速览（2020年 第3期）

发表于 2020-03-08 | 分类于 随笔 | | 阅读次数：

时间：2020年3月1日-3月8日

本周大概就是工作占据了特别多时间吧，所以在开这个日志的时候，感觉到没有输出了。没想到这种感觉来得如此快，与此同时，本着避免把速览写成流水账的初心，怎么在碎片的「知识点」间找到连接点是很有挑战的一件事，但我愿意尽力试一试：

you can’t connect the dots looking forward,
you can only connect them looking backwards.
So you have to trust the dots will somehow connect in your future.

工作时间，目前都是我无奈的「浪费时间」：整个一周都在看巴塞尔协议III衍生出的央行对商业银行监管的MPA考核体系、银保监的《商业银行风险监管核心指标》以及《商业银行内部指引》，合起来简称监管机构的“两项评级”内容，boring thing。

不过，倒是加深了一个认识：工作是会占据人生的非常多时间的，即便不加班的朝九晚五，1/3时间休息，剩下16个小时至少有8个小时给了工作。实际上，通勤以及围绕「去工作」花掉的时间至少10小时！实在是没有什么理由做一份完全只是为了钱的工作。想起来比较认同一个观点：如果觉得需要每个周末来唤醒生活，周一到周五都是行尸走肉，是时候需要好好思考下你正在做的事情，到底值不值得。
